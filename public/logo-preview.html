Instructions to create favicon files:

  1. Use an online tool like Favicon.io or Real Favicon Generator
  2. Upload the SVG file (datalense-logo.svg)
  3. Generate the favicon.ico, logo192.png, and logo512.png files
  4. Replace the existing files in the public directory

Note: You can also use graphic design software like Adobe Illustrator, Figma, or Inkscape to make adjustments to the SVG before converting.
